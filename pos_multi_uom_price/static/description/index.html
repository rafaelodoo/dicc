POS Multi UOM Price

With this module you can sell your products with different units of measure in POS.

Features

  • Configure multiple UOM prices per product
  • Variant-level price control

Setup

Activate the Units of Measure setting

  • Inventory > Configuration > Settings
    • Set "Units of Measure"

Install the "POS multi uom price" module

Configure multiple UOM prices

  • Inventory > Products > Products
    • Product (Sales)
      • Add "UOM price"

Description

Unit prices for products can be configured in the "Point of Sale" tab of each product.

If product variants are enabled, the unit prices defined in the product template will be automatically replicated to the variants.

  • If you update the unit price in the template, the changes will be reflected in all variants.
  • However, if you update the unit price directly in a variant, this change will not be propagated back to the template.
  • Important: The Point of Sale (PoS) will only load the unit prices defined in the variants, not those from the template.
